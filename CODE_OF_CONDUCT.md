# Code of Conduct

## 私たちの約束

FireSohouAIは、消防関係者、研究者、開発者、学生、初心者など、立場や経験を問わず参加できる環境を目指します。

## 期待する行動

- 相手の経験と専門性を尊重する
- 安全性を最優先する
- 根拠と検証方法を示す
- 間違いを責めず、改善につなげる
- 個人情報・所属情報を慎重に扱う
- 現場の慣習と技術的合理性の両方を尊重する
- 公式審査と本プロジェクトの参考評価を混同しない

## 認められない行動

- 差別、侮辱、威圧、嫌がらせ
- 本人の同意がない映像や個人情報の公開
- 安全上危険な手順の推奨
- 根拠のない断定的な採点
- 他者の成果やデータの無断利用
- プロジェクトを公式機関の認定品であるかのように表示する行為

## 対応

問題を発見した場合は、公開Issueに個人情報を書かず、リポジトリ管理者へ連絡してください。
