# Roadmap

## 目的

1年後の操法大会への参加を見据え、現場訓練とソフトウェア開発を連動させます。

## Phase 0: プロジェクト基盤

目安: 2026年7月

- [x] GitHubリポジトリ作成
- [x] 初期ディレクトリ構成
- [x] README、ライセンス、参加ガイド
- [ ] GitHub Projects作成
- [ ] 初期Issue登録
- [ ] ブランチ・レビュー方針の確定

## Phase 1: デジタル教範

目安: 2026年7月〜9月

- [ ] 小型ポンプ操法のシーン分割
- [ ] 教材用映像プロンプト
- [ ] AI解析用映像プロンプト
- [ ] カメラ配置設計
- [ ] 教範映像の試作
- [ ] 現場関係者によるレビュー

成果物: 理想動作の映像設計と基準データ

## Phase 2: 撮影基盤

目安: 2026年8月〜11月

- [ ] Raspberry Pi 5撮影環境
- [ ] Pi Camera設定
- [ ] 60fps録画試験
- [ ] ファイル命名規則
- [ ] 撮影メタデータ保存
- [ ] 複数カメラ同期の検討
- [ ] 屋外電源・冷却・耐候性の確認

成果物: 再現可能な訓練映像撮影手順

## Phase 3: 姿勢・器具解析

目安: 2026年9月〜2027年1月

- [ ] MediaPipe姿勢推定
- [ ] 肘・膝・腰・体幹角度
- [ ] 移動軌跡・歩幅・歩数
- [ ] YOLOデータセット設計
- [ ] ホース・筒先・吸管・ポンプ等の認識
- [ ] 動作イベント検出
- [ ] 解析結果CSV出力

成果物: 姿勢・器具・動作イベント解析のプロトタイプ

## Phase 4: 自動計測・比較

目安: 2026年12月〜2027年3月

- [ ] 操作開始の検出
- [ ] 役割別ラップ計測
- [ ] 放水開始・標的倒壊の検出
- [ ] 理想動作との時間比較
- [ ] 姿勢差・動線差の可視化
- [ ] 誤検出の確認方法

成果物: 訓練後に確認できる比較レポート

## Phase 5: 現場運用

目安: 2027年1月〜大会前

- [ ] 訓練ごとの撮影・解析
- [ ] GitHub Issueへの現場課題登録
- [ ] 解析結果を次回訓練へ反映
- [ ] 個人別・チーム別の推移確認
- [ ] 安定版リリース候補作成
- [ ] 大会直前版の動作確認

成果物: Competition Edition Release Candidate

## Phase 6: 大会後

- [ ] 実運用の振り返り
- [ ] 効果と限界の整理
- [ ] 公開可能データの選別
- [ ] Version 1.0リリース
- [ ] 次年度ロードマップ
- [ ] 他訓練への展開検討

## 判断基準

新機能は次の順で優先します。

1. 安全性
2. 現場での必要性
3. 測定可能性
4. 再現性
5. 実装コスト
6. 公開・再利用のしやすさ
