# Master Prompt

## 目的

全シーンで人物、会場、器具、映像条件を統一する。

## 共通条件

- 全国大会教範映像風
- 小型ポンプ操法
- 実写的で自然な映像
- 過度な映画演出なし
- 自然光
- 人物と器具の外観を維持
- 余分な人物や器具を追加しない
- 文字、ロゴ、透かしを追加しない

## 教材用

- 動作の流れが理解しやすい
- 原則固定カメラ
- 主要動作が隠れない
- 視聴者向けの分かりやすさを優先

## AI解析用

- 全身が画面内
- 人物同士の重なりを最小限にする
- 手足と器具の輪郭が明瞭
- 背景を単純化
- 固定カメラ
- モーションブラーを抑える

## 注意

生成映像は教範そのものではない。具体的な動作は実施要領と現場レビューで確認する。
