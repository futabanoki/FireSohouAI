# Storyboard

小型ポンプ操法の教範映像とAI解析用映像のシーン設計を管理します。

## 各シーンに含める内容

- シーン名
- 目的
- 根拠となる実施要領
- 開始状態
- 秒単位タイムライン
- 教材用カメラ
- 解析用カメラ
- 教材用プロンプト
- 解析用プロンプト
- 終了フレーム
- MediaPipe解析項目
- YOLO認識項目
- 手動確認項目
- 既知の生成上の問題

## ファイル名

```text
scene01_waiting_and_report.md
scene02_operation_start.md
scene03_suction_and_pump_setup.md
```
