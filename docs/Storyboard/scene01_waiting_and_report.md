# Scene 01: 待機・点呼・開始報告

## 状態

Draft

## 目的

- 4名の外観を固定する
- 会場・器具・カメラ条件を固定する
- 教範映像の開始場面を示す
- 次シーンへ連続する終了姿勢を作る

## 長さ

15秒を基本とする。

## 教材用カメラ

- 固定三脚
- 全員と主要器具を表示
- 教範映像として動作が分かる斜め前方または正面
- ズーム、パン、チルトなし

## 解析用カメラ

- 固定三脚
- 全身表示
- 人物同士の重なりをできるだけ避ける
- 背景を単純化する
- フレームレートとシャッター条件を記録する

## タイムライン案

- 0〜2秒: 待機
- 2〜5秒: 点呼開始
- 5〜8秒: 各隊員の応答
- 8〜11秒: 開始報告
- 11〜13秒: 想定付与
- 13〜15秒: 次動作の基本姿勢

## 解析項目

- 号令から反応までの時間
- 基本姿勢
- 頭部・肩・腰の向き
- 整列位置
- 次シーン開始時の同期性

## 注意

具体的な号令、姿勢、移動、器具配置は、採用する大会要領の原文と現場レビューに基づいて確定する。
