# Camera Design

## カメラ種別

### Camera A: 教材用

- 視聴者が動作の流れを理解しやすい構図
- 原則固定
- 器具と人物の関係を見せる
- 過度な演出を避ける

### Camera B: AI解析用

- 関節点と器具が隠れにくい構図
- 固定
- 全身表示
- 背景を単純化
- 撮影条件を毎回記録

### Camera C: 現場実演用

- Raspberry Piまたは市販カメラ
- 実際の訓練を撮影
- Camera Bと近い条件を優先
- 設置位置を地面マーキング等で再現可能にする

## 記録項目

- camera_id
- 設置位置
- 高さ
- 方位
- 被写体までの距離
- 解像度
- fps
- シャッター速度
- レンズ・画角
- 日時
- 天候
- 照明
