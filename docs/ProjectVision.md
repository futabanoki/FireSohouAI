# Project Vision

## Mission

消防操法の教育・訓練・研究を支援するため、現場知識とAI技術を結び付けたオープンソース基盤を開発します。

## Vision

誰もが利用・検証・改善に参加できる、透明性と再現性のある消防操法AI支援環境を構築します。

## 解決したい課題

- 動作の改善点が感覚的な説明だけになりやすい
- 複数隊員のタイミングを同時に確認しにくい
- 訓練映像の比較に時間がかかる
- 姿勢や動線の変化を継続的に残しにくい
- 現場知識とAI開発が分断されやすい

## AIの役割

AIは、次のような客観情報を提供する補助役です。

- 動作時刻
- 姿勢と関節角度
- 移動軌跡
- 器具の位置
- 理想モデルとの差
- 訓練ごとの変化

最終的な判断、採点、指導は人が行います。

## 初期利用者

- 操法大会を目指す消防団チーム
- 訓練指導者
- 消防操法を研究する人
- コンピュータビジョン開発者
- Raspberry Piを使った撮影・解析の開発者

## 成功の定義

- 現場で継続利用できる
- 解析結果を次回訓練に反映できる
- 測定条件と限界が明記されている
- 他のチームが再現できる
- 個人情報と安全性が守られる
- 大会後にも知識とコードが資産として残る
