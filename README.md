# FireSohouAI
An open-source AI platform for firefighting drill training, analysis, and education.
This project aims at the nationwide top team adhering to the latest offfical textbook about "Yonin-souhou"四人操法 consisting of 4 person work with a small fire pomp, 3 hoses and a playpipe.
This is written by a traiing manager of 8th divison of Kanaza-Shouboudan"金沢消防団第8分団" in Yokohama, Kanagawa Japan
